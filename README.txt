Corporate Practice Website Library

Files included:
- index.html
- style.css
- script.js
- assets/ (SVG illustrations used by the site)

How to use:
1. Keep all files in the same folder structure.
2. Open index.html in a browser.
3. For best results, do not move the assets outside the assets folder.

Design notes:
- Light mode only
- Minimalist presentation-style layout
- Fully offline-ready except for Google Fonts, which can be removed if needed
